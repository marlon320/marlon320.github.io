Link to design document 
              Game overview: Inspired by the game drawful two. Who's Your Daddy aims to encourage the use
              of Cherokee in a public setting. This multiplayer game will promt each player to create a funny
              question. Then one question is is selected at random. Each player will create a funny responce to 
              the question. Players will then vote on whos response was the funniest. Then the person who wrote
              the question and the winner will say their question/response. This encourages the use of speaking 
              Cherokee with public players. The players earn points based on pronunciation and winning votes.
